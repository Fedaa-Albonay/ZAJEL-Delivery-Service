🚚 ZAJEL Delivery Service Website
A modern and responsive delivery service website built as a front-end project to simulate a real logistics platform in the UAE.

📌 Overview
ZAJEL is a web application designed to represent a delivery and logistics service.
It provides users with an intuitive interface to explore services, track shipments, and communicate easily with the support team.
This project was built as my first Software Engineering project, focusing on front-end development and user experience.

✨ Features
    • 🔹 Responsive and modern UI design
    • 🔹 User authentication (Login & Registration using Local Storage)
    • 🔹 Shipment tracking system (with validation logic)
    • 🔹 Contact form with validation and success feedback
    • 🔹 Dark mode toggle 🌙
    • 🔹 Smooth scrolling and interactive UI elements
    • 🔹 Phone input with country code support

🛠️ Technologies Used
    • HTML5 – Structure of the website
    • CSS3 – Styling and layout (Flexbox & Grid)
    • JavaScript (Vanilla JS) – Functionality and interactivity
    • Font Awesome – Icons
    • intl-tel-input – Phone number input handling
